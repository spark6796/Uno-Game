# 🃏 UNO — Multiplayer Card Game

Play UNO with your friends in real-time!  
Create a room, share the code, and let the battle begin. Powered by WebSockets for smooth live gameplay.

## 🚀 Run Locally

### Backend (FastAPI)
```bash
uv run fastapi dev
```

### Frontend (Svelte)
```bash
npm run dev
```
